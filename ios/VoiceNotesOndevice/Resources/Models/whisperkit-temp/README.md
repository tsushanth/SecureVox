# WhisperKit

WhisperKit is an on-device speech recognition framework for Apple Silicon:
https://github.com/argmaxinc/WhisperKit

For performance and accuracy benchmarks on real devices, please see:
https://huggingface.co/spaces/argmaxinc/whisperkit-benchmarks

WhisperKit Pro is the commercial upgraded version of WhisperKit. If you would like to learn more about upgrading to Pro, please reach out to whisperkitpro@argmaxinc.com or fill the form [here](https://huggingface.co/argmaxinc/whisperkit-pro).
